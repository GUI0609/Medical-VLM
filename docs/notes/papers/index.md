## 示例卡片

| 字段 | 内容 |
| --- | --- |
| 问题 | 医学视觉语言模型如何提升报告生成或视觉问答？ |
| 方法 | 对齐图像编码器和文本生成模型，使用任务数据微调。 |
| 证据 | 数据集、指标、消融、失败样例是否充分。 |
| 决策 | 值得复现、只做背景阅读、或暂时跳过。 |
