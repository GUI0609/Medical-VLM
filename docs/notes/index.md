# 学习笔记

本区按主题沉淀可公开的学习材料。未整理、含隐私或未确认是否可公开的材料先放在本地 `inbox/`，不得直接提交。

## 分类

- [AI](ai/index.md)
- [编程](programming/index.md)
- [论文](papers/index.md)
- [科研](research/index.md)

## 笔记模板

建议每篇笔记保持固定结构：

1. 问题：这篇笔记回答什么问题。
2. 结论：当前可以支持的判断。
3. 证据：数据、代码、论文或实验来源。
4. 风险：结论边界和不确定点。
5. 下一步：继续投入或停止的决策。
